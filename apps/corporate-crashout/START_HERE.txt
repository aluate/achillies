QUICK START (5 Steps):
----------------------

1. INSTALL NODE.JS
   - Go to: https://nodejs.org/
   - Download LTS version (big green button)
   - Install it (accept defaults)
   - RESTART YOUR COMPUTER (important!)
   - Verify: Open Command Prompt, type "node -v"
   - Should see version number (like v20.x.x)

2. OPEN PROJECT FOLDER
   - Find the "corporate-crashout" folder
   - Open File Explorer to that folder
   - Click in address bar, type "cmd", press Enter
   - Command Prompt opens in that folder

3. INSTALL DEPENDENCIES
   - In Command Prompt, type: npm install
   - Press Enter
   - Wait 2-5 minutes (lots of text scrolling)
   - Done when cursor comes back

4. SET UP DATABASE & ENV FILE
   - I'll help you set up a free database (Supabase)
   - Create .env file (copy from .env.example if it exists)
   - Fill in DATABASE_URL (I'll send you this)
   - Add Stripe test keys (I'll send these too)

5. BOOTSTRAP & START
   - Type: npm run bootstrap
   - Wait for it to finish
   - Write down admin credentials it shows
   - Type: npm run dev
   - Open browser to: http://localhost:3000
